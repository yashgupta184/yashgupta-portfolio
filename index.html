Marvel Tic-Tac-Toe

Project Overview

“Marvel Tic-Tac-Toe” is a themed version of the classic Tic-Tac-Toe game featuring Thor’s hammer and Captain America’s shield instead of the usual X and O. The game tracks each player’s wins and losses until the “universe” (game) is reset, adding a fun, continuous competitive element. It combines engaging visuals with persistent scoring to enhance user experienc

Key Features

  • Themed gameplay using Thor’s hammer and Captain America’s shield as markers
  • Reset option to start a new “universe” while clearing scores
  • Tracks individual player wins and losses across multiple rounds
  • Interactive, user-friendly interface with responsive design
